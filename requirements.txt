## Troubleshooting

### Error: "No module named 'requests'"
```powershell
pip install requests
```

### Error: "No module named 'numpy'"
```powershell
pip install numpy
```

### Error con Flask en Windows
```powershell
pip install --upgrade flask flask-cors
```
